# TRL0016 — PR title/description standard, template, and draft shortcut

## Scope
- `.github/pull_request_template.md` defining the description sections and the
  `TRLxxxx — <imperative summary>` title convention.
- A draft shortcut (`docs/playbooks/pr-draft.md`): a ready prompt that turns the
  branch's commits/diff into a copy-ready PR title + description conforming to
  the template.
- Enforce the standard in CI (e.g. a PR-title lint on pull_request).
- Graduate the shortcut to an MCP prompt so any tool can invoke it (with TRL0006).

## Notes
- The template and playbook ship alongside the TRL0015 PR. CI enforcement and the
  MCP-prompt graduation are the remaining scope, so this stays active until those
  land — cleanest once the TRL0001 generator foundation has merged.

## Risks
- Title-lint false positives on legitimate multi-item PRs; allow a primary-id
  lead with the rest named in the body.
