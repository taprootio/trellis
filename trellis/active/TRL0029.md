# TRL0029 — Importer: strip the source id prefix from imported titles

## Scope
- In the importer's title resolution (`planImport`, where the item's `sourceId` is
  already resolved), when the resolved title begins with the item's **own** source id
  followed by a separator, drop that leading token from the stored `title` — e.g.
  `# 001 README Truth Pass` under source id `001` → `README Truth Pass`. The same
  cleaned title heads the rebuilt body (`# <newId> — <title>`); body prose is untouched.
- Match exactly the item's resolved source id, then an optional single separator
  (`.` `:` `-` `–` `—`), but only when a whitespace break follows. This is conservative
  by construction: id `04` never bites into `047 …`, `001README` (no break) is left
  intact, `001 .NET` keeps its dot, and a genuinely number-leading title (`2024 Roadmap`
  under a different id) stays. Never blank a title that is nothing but the id.
- Drive it from the mapping via a `stripIdPrefix: true` flag on the `title` field —
  **explicit, not inferred**: default off, set on for the numeric-prefix
  `taproot-ai-backlog` profile.

## Notes
- Dogfood friction (TRL0009): every imported Taproot title kept its old number
  ("001 README Truth Pass", "047 Design the Espalier MCP Server") because the `h1`
  extractor takes the heading verbatim. Cosmetic, but it shows on all imported items.
- Refinement decisions: the flag is **explicit in the mapping** (the engine never
  infers it from a numeric id pattern — it stays free of heuristics about what an id
  extractor "means", matching the importer's "never a silent guess" ethos); the
  separator match accepts **whitespace and/or one** of `. : - – —`.
- Cosmetic, title-content only — no SPEC change (SPEC only requires `title` be one
  line). No new front-matter field; the change is the importer's, gated by the profile.

## Risks
- Be conservative — strip only the leading token that exactly matches the item's source
  id, never a meaningful leading word or number, to avoid mangling real titles. The
  exact-id match plus the required whitespace break is what delivers that guarantee;
  cover it with a guard fixture (a real number-leading title that must survive).
