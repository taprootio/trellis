# TRL0005 — Build the local viewer (`trellis serve`)

## Scope
- Watch the backlog root (`tasksDir`, default `trellis/**`), regenerate, and
  serve a single-file SPA over `backlog.json`.
- Live-reload the browser on change via SSE.
- Board grouped by milestone; click through to the task file.

## Notes
- v1 is local-only by design. Deployment and SSO are deferred (TRL0012, TRL0013).

## Risks
- Watch loops thrashing on generator writes. Debounce and ignore generated files.
