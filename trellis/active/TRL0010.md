# TRL0010 — Packaging, versioned spec, and docs

## Scope
- Publishable package; `npx trellis init|serve` and the MCP server entrypoint.
- Versioned `SPEC.md` with a compatibility note.
- Getting-started + reference docs.
- Dogfood the packaged onboarding: re-run TRL0009's recipe against a real repo using
  the published `npx trellis` (init → import → generate → check) with the scaffolded
  CI gate actually green — proving the consumer onboarding path, not just the format
  (TRL0009 proves format/process on dev tooling; this proves the package).

## Risks
- Semver discipline on the spec vs. the tooling. Version them independently.
